## üõ†Ô∏è Technologies Used

- **PHP**, **HTML5** and **CSS3**: Raw php 7.4 (updated to 8.3) for the main structure, dinamically generating the HTML templates and using CSS for responsive and clean styles.

- **datatables**: DataTables is a Javascript HTML table enhancing library. It helps with database data sorting and presentation.

- **fpdf**: PHP class that allows the generation of pdf documents directly from PHP, avoiding PDFlib library. Free to use.

- **mysql**: SQL database for data management and storage. Easily accesed via PHPMyAdmin.  

- **ajax**: It's mostly ajax based, generating dynamic tables from user inputs.
